# TutorMatch
TutorMatch is a startup that helps students find the best tutors all around the world. 

Available Features:
  - Retrieve tutors
  - Create a new tutor
  - Update any tutor information
  - Delete tutors 
 
### Running Application
TutorMatch requires [Node.js](https://nodejs.org/) v8+ to run.
  - Download and install Nodejs
  - Navigate to application directory and install dependencies
  
` Nodejs installation is required`

Install the dependencies and start application.

```
$ npm install 
$ npm run start
```
### Browser Test
  - Launch the browser and navigate to ``localhost:4200``

### Running Tests
```
npm run test
```
